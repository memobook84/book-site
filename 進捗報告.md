## 完成した機能

### トップページ（index.html）
- 90作品の漫画を6列×15行のグリッドで表示
- 各作品をカラフルな色分けで表示（CSSベース）
- 検索機能（タイトル・著者名で検索可能）
- ヘッダーナビゲーション（New Release・Followへのリンク）

### 作品詳細ページ（detail.html）
- 作品情報の表示（タイトル、著者、出版社、レーベル、ジャンル、発売日、あらすじ）
- 著者リンク（クリックで著者ページへ遷移）
- フォロー機能（ハートアイコンでお気に入り登録、フォロー時に赤く変化）
- 各巻のリスト表示（クリックで巻詳細ページへ）
- 各巻の発売日を自動計算（初回発売日から3ヶ月ごと）

### 巻詳細ページ（volume.html）
- 各巻の詳細情報
- 購入ボタン（楽天・Amazon）
- 作品ページへの戻るリンク

### 著者ページ（author.html）
- 著者の略歴表示
- 代表作品リスト
- その著者の全作品をグリッド表示

### フォローページ（follow.html）
- フォロー中の作品をグリッド表示
- フォロー解除機能
- localStorageでデータ永続化
- 空の状態のメッセージ表示

### 新刊ページ（new-releases.html）
- 最新20作品をリスト形式で表示
- NEW バッジ表示（上位3作品）
- 発売日順にソート
- クリックで作品ページへ遷移

## データベース

### manga-data.js
- 90作品の詳細データベース
- 実在する人気漫画タイトルを使用
- 各作品のデータ項目：
  - id, title, author, publisher, label, genre
  - firstReleaseDate, description, totalVolumes, color
- カラーパレット（82色）

- 僕のヒーローアカデミア, ハイキュー!!, 進撃の巨人, HUNTER×HUNTER
- その他、人気漫画84作品

## デザイン仕様

### 2/16 UIリデザイン（Barnes & Noble風 → カスタムデザイン）

#### カラースキーム
| 用途 | 色 |
|---|---|
| ヘッダー背景 | `#595959`（ダークグレー） |
| ページ背景 | `#FAF8F5`（温かみのあるクリーム） |
| カード背景 | `#FFFFFF` |
| テキスト（主） | `#2B2B2B` |
| テキスト（副） | `#5A5A5A` |
| リンク色 | `#6cb4ee`（薄い青） |
| ボーダー | `#E5E1DC`（温かいグレー） |
| フォローハート（未） | `#999`（グレー） |
| フォローハート（済） | `#e74c3c`（赤） |
| 楽天ボタン | `#8B6B5C`（ウォームブラウン） |
| Amazonボタン | `#A68A6F`（トープ） |
| NEWバッジ | `#9B6B6B` |

#### フォント
- **ヘッダー（英語）**: Inter（アルファベットのみ）
- **見出し（英語）**: Crimson Pro（アルファベットのみ）
- **日本語・数字**: Noto Sans JP（全て）
- `@font-face` の `unicode-range` でアルファベットのみInter/Crimson Proを適用

#### ヘッダー
- 背景: ダークグレー（`#595959`）、文字: 白
- コンパクトなパディング（`7px`）
- 下線: `1px solid #444`
- ナビリンク: 大文字表記（NEW RELEASE, FOLLOW）
- 検索窓: `Search manga...`（英語）

#### デザイン方針
- シンプルで自然なデザイン
- 枠線や影を削除（AI臭を排除）
- 余白を活用したクリーンなレイアウト
- ホバーエフェクトで適度なインタラクション
- カードに白背景+薄いシャドウ（トップページのグリッド）

### ファビコン
- 惑星アイコン（`favicon.png`）を全ページに設定

## 技術仕様

### フロントエンド
- HTML5 / CSS3 / JavaScript (Vanilla JS)
- レスポンシブデザイン対応
- Google Fonts使用（Inter, Playfair Display, Crimson Pro, Noto Sans JP）
- CSS変数（カスタムプロパティ）による一元管理

### データ管理
- 共通データベース（manga-data.js）
- localStorage（フォロー機能）

### 画像表示
- CSSベースのプレースホルダー
- 各作品固有の色で表示
- 外部サービスに依存しない（完全オフライン動作可能）

### サーバー
- Python HTTP Server（開発用）
- ポート: 8000
- URL: http://localhost:8000

## 2/16の作業内容（UIリデザイン）

### 実施した変更
1. **フォントリンク更新** - 全6HTMLファイルにInter, Playfair Display, Crimson Pro追加
2. **CSS変数定義** - style.cssに`:root`でカラー・フォント変数を一元管理
3. **ヘッダーリデザイン** - ダークグレー背景、コンパクト化、英語ナビリンク（大文字）
4. **カラーパレット変更** - 全体をクリーム背景+落ち着いた色調に統一
5. **フォント分離** - `@font-face` unicode-rangeでアルファベットのみInter/Crimson Pro、日本語・数字はNoto Sans JP
6. **リンク色統一** - 全ページのリンクを薄い青（`#6cb4ee`）に変更
7. **フォロー機能** - ハートアイコン、フォロー時に赤く変化
8. **ファビコン設定** - 惑星アイコンを全6ページに設定
9. **枠の削除** - 作品ページ等のカード枠を削除（AI臭排除）
10. **テキスト色修正** - 薄くて見えにくかったテキストを黒に修正
